# 白噪音哄睡小程序 · 项目记忆

## 项目概览
- **类型**: 微信小程序
- **功能**: 白噪音播放 + 婴儿哭声识别
- **目标用户**: 母婴人群

## 技术架构
- 原生微信小程序（无框架）
- 组件: `tips-marquee`, `cry-detector`, `sound-card`
- 音频: `wx.getBackgroundAudioManager()` 全局播放
- 哭声识别: 调用 FC 函数识别（录音 → base64 → 请求）

## UI 风格
- **母婴极简风**: 莫兰迪粉 + 莫兰迪绿 + 米白底色
- **大圆角**: 24-32rpx
- **柔和阴影**: 粉色系阴影
- **4列网格**: 白噪音卡片
- **图标**: 线框风格，透明背景

## 关键文件
- `config/sounds.js` - 声音配置（id, name, icon, url）
- `app.js` - 全局音频管理 + 哭声识别配置
- `components/sound-card/` - 白噪音卡片组件
- `components/cry-detector/` - 哭声识别组件
- `images/sound-icons/` - 9个声音图标（正常态 + 播放态）
